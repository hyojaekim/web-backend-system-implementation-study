# AWS S3(Simple Storage Service)

## 소개
* 다양한 오브젝트를 REST 인터페이스를 통해 저장/삭제/조회 할 수 있다.
  * 서버가 2대 이상 분산된 경우 이미지 등을 처리하기 좋다.
* 99.99% 이상 내구성 제공
* 데이터 무제한 저장 가능 (개별 오브젝트 별 최대 5TB까지 저장 가능)
* 오브젝트에 대한 접근 제어 가능

## 주요 용어
* 객체(Object): S3에 데이터가 저장되는 최소 단위, 실제 파일과 메타데이터로 구성
  * 키(Key): 객체 이름, 값(value): 객체의 데이터
* 메타데이터: HTTP Content-Type 형식 MIME 타입으로 표현
* 버킷(bucket): S3에서 생성할 수 있는 최상위 폴더 (해당 위치에 객체 저장)
* 리전(Region): 객체가 저장되는 물리적인 위치, 리전 간 객체공유 불가능
  * 지연 시간 최적화, 비용 최소화, 규정 요구 사항 준수 등 다양한 필요에 따라 리전 선택
